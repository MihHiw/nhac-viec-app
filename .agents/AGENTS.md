## Quy tắc làm việc mới

**Luôn lập kế hoạch trước khi code:** Đối với mọi yêu cầu của User, Agent BẮT BUỘC phải tạo một bảng Kế hoạch (Implementation Plan) và giải thích rõ ràng. Agent chỉ được phép tiến hành viết code hoặc thay đổi hệ thống SAU KHI User đã nhấn nút "Proceed / Đồng ý" trên bảng Kế hoạch đó. Tuyệt đối không tự ý sửa code khi chưa có sự cho phép.
